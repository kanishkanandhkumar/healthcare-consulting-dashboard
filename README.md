Healthcare Consulting Dashboard
This project analyzes a sample patient dataset to identify patterns in readmissions, medication usage, demographics, and lab values. The Excel file includes data cleaning, calculated fields, pivot tables, and a final dashboard.

 Workflow
-Cleaned and structured patient data
-Created calculated fields for risk groups, lab ranges, age segments, and readmission status
-Built pivot tables for admissions, demographics, medications, and lab metrics
-Designed an interactive dashboard with slicers

Dashboard Highlights
Monthly admissions
-Readmission distribution
-Demographic overview
-Medication usage summary
-Lab value segmentation

Skills Demonstrated
Excel cleaning, formulas, pivot tables, dashboard design, and healthcare data analysis.
